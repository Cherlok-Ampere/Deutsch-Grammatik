01

Die vier Fälle

Nominativ · Akkusativ · Dativ · Genitiv

Nominativ

Wer? Was?

Das Subjekt des Satzes — derjenige, der die Handlung ausführt.

Der Hund bellt laut. / Eine Frau liest das Buch.

Akkusativ

Wen? Was?

Das direkte Objekt — das, worauf die Handlung direkt gerichtet ist.

Ich sehe den Hund. / Er kauft einen Apfel.

Dativ

Wem?

Das indirekte Objekt — der Empfänger oder Begünstigte der Handlung.

Ich gebe dem Kind ein Buch. / Sie hilft einer Freundin.

Genitiv

Wessen?

Drückt Zugehörigkeit oder Besitz aus. Im Alltag oft durch Dativ ersetzt.

Das Auto des Mannes. / Die Farbe des Himmels.
⚡ Merktrick Nominativ = Subjekt (Täter). Akkusativ = direktes Objekt (Opfer der Handlung). Dativ = indirektes Objekt (Empfänger). Genitiv = Besitz (von wem?). Die Frage «Wer tut was wem?» deckt die drei häufigsten Fälle ab.

03

Adjektivdeklination

Stark · Schwach · Gemischt

📌 Grundregel Das Adjektiv trägt die Kasusendung, wenn der Artikel sie nicht zeigt (starke Deklination). Wenn der Artikel die Endung schon trägt (bestimmter Artikel), wird das Adjektiv «geschwächt» (nur -e oder -en).

+ Akkusativ

Nur Akkusativ

gegen gegen + Akk. = entgegen / dagegen

Perfekt & Präteritum

Passé composé · Imparfait / Passé simple

📌 Wann welches Tempus? Perfekt = gesprochene Vergangenheit, Alltagssprache (« j'ai fait »). Präteritum = schriftliche Vergangenheit, Erzählungen, Literatur (« je fis / je faisais »). Ausnahme: sein, haben, werden und Modalverben → immer Präteritum, auch mündlich.
Perfekt

Das Perfekt — haben oder sein?

haben + Partizip II

Die meisten Verben — transitiv oder ohne Bewegung/Zustandsänderung.

Ich habe gegessen.
Er hat gespielt.
Wir haben gearbeitet.
sein + Partizip II

Verben der Bewegung / Zustandsänderung + sein, bleiben, werden.

Ich bin gegangen.
Er ist aufgestanden.
Sie ist eingeschlafen.
Bildung des Partizip II
TypSchemaBeispiele
Regelmäßig (schwach) ge- + Stamm + -t spielen → gespielt · kaufen → gekauft · arbeiten → gearbeitet
Unregelmäßig (stark) ge- + Stamm (verändert) + -en fahren → gefahren · schreiben → geschrieben · singen → gesungen
Trennbare Verben Präfix + ge + Stamm + -t/-en aufmachen → aufgemacht · einschlafen → eingeschlafen
Untrennbare Verben Kein ge- ! besuchen → besucht · verstehen → verstanden · erklären → erklärt
Verben auf -ieren Stamm + -t (kein ge-) studieren → studiert · telefonieren → telefoniert

07

Das Passiv

Vorgangspassiv · Zustandspassiv

📌 Was ist das Passiv? Im Passiv steht die Handlung im Vordergrund, nicht der Handelnde. Das Objekt des Aktivsatzes wird zum Subjekt. Der Handelnde kann mit von + Dativ angegeben werden.
Aktiv

Der Koch kocht die Suppe.

Subjekt (Koch) = Handelnder

Passiv

Die Suppe wird (vom Koch) gekocht.

Subjekt (Suppe) = betroffenes Objekt

Vorgangspassiv — werden + Partizip II (alle Zeiten)
TempusBildungBeispiel
Präsens wird + Partizip II Das Buch wird gelesen.
Präteritum wurde + Partizip II Das Buch wurde gelesen.
Perfekt ist + Partizip II + worden Das Buch ist gelesen worden.
Plusquamperfekt war + Partizip II + worden Das Buch war gelesen worden.
Futur I wird + Partizip II + werden Das Buch wird gelesen werden.
werden — Konjugation Präsens & Präteritum

Zustandspassiv — sein + Partizip II (Ergebnis einer Handlung)
TempusBildungBeispiel
Präsens ist + Partizip II Das Geschäft ist geschlossen. (= es ist jetzt zu)
Präteritum war + Partizip II Das Geschäft war geschlossen.
Vorgangspassiv

Die Tür wird geschlossen.

→ Jemand schließt sie gerade (Prozess)

Zustandspassiv

Die Tür ist geschlossen.

→ Die Tür befindet sich im Zustand "zu" (Resultat)

Passiv mit Modalverben — müssen / können / sollen + Partizip II + werden
AktivPassiv
Man muss das Formular ausfüllen.Das Formular muss ausgefüllt werden.
Man kann das Problem lösen.Das Problem kann gelöst werden.
Man soll den Müll trennen.Der Müll soll getrennt werden.
⚠️ Achtung: worden vs. geworden Im Passiv-Perfekt: worden (ohne «ge-»): Das Haus ist gebaut worden.
Das Perfekt von werden (aktiv): ist geworden: Er ist Arzt geworden.

📌 Die drei Stufen Positiv (Grundstufe) → Komparativ (mehr als) → Superlativ (das Meiste). Alle drei können als Adjektiv (mit Deklination) oder als Adverb (unveränderlich) stehen.
